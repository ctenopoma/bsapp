# 特許調査（Patent Research）入力ガイド

このファイルを編集して、特許調査ヘルパーの知識を追加してください。

## 基本的な考え方

Patent Researchは、CSVファイルから特許情報を読み込み、企業ごとの技術動向レポートを自動生成する機能です。

## 分析設定の各項目

- **企業別分析 システムプロンプト**: 各企業の特許を分析する際のAIへの指示。「〇〇の観点から分析してください」など目的を具体的に書く
- **企業別分析 出力フォーマット**: レポートの形式を指定。Markdownの見出し・箇条書きで構造を定義すると整った出力になる
- **総括 システムプロンプト**: 全企業を横断してまとめる際の指示。「競合比較」「技術トレンド」など総括の切り口を指定する

## システムプロンプトの書き方

良い例：
```
あなたは特許分析の専門家です。提供された特許リストから、この企業の技術戦略と注力分野を分析し、
今後の技術動向を予測してください。競合他社との差別化ポイントにも言及してください。
```

## 出力フォーマットの例

```
## {company} 技術動向レポート

### 注力技術分野
-

### 主要特許のポイント
-

### 技術トレンドの考察

### 今後の方向性予測
```

## CSV列名の設定

CSVの列名はSettings画面で設定します。J-PlatPatからエクスポートした場合のデフォルト値：
- 企業列: `出願人`
- 内容列: `発明の名称`
- 日付列: `出願日`
