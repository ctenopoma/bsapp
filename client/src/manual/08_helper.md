# ヘルパー機能

**New Session・Personas・Tasks** の各ページで利用できるAIアシスタント機能です。
入力項目の書き方に迷ったとき、AIにチャット形式で相談できます。

## ヘルパーの開き方

各ページの右下に表示される **ヘルパー** ボタン（紫色・チャットアイコン）をクリックします。

## ヘルパーでできること

| 機能 | 説明 |
|------|------|
| **質問・相談** | 入力項目の書き方や考え方をチャットで相談 |
| **提案の生成** | 現在の入力内容を踏まえた具体的な記入案を提示 |
| **提案の反映** | AIが提案した内容をフォームに直接反映 |
| **コピー** | 提案テキストをクリップボードにコピー |

---

## 使い方

### 1. 質問する
テキストエリアに質問を入力して **送信（Enterキー）** または **送信ボタン** を押します。

**質問の例：**
- 「どんなロールを書けばいい？」
- 「批評家のペルソナを作りたい」
- 「ストラテジーの違いを教えて」
- 「新製品の企画会議に合うテーマを提案して」

### 2. 提案を確認する
AIの返答が表示されます。具体的な提案がある場合は **提案カード** も表示されます。

### 3. 提案をフォームに反映する

提案カードには以下のボタンがあります：

| ボタン | 説明 |
|--------|------|
| **コピー（📋アイコン）** | その項目のテキストをクリップボードにコピー |
| **貼り付け（📌アイコン）** | その項目だけをフォームに反映 |
| **まとめて反映** | 全提案項目をまとめてフォームに反映 |

### 4. 自動反映モード（自動ON/OFF）

ヘッダーの **自動OFF/ON ボタン** をクリックすると自動反映モードに切り替わります。

- **自動ON（黄色）** : AIが提案を出すたびに自動的にフォームに反映されます
- **自動OFF（デフォルト）** : 手動で反映ボタンを押す必要があります

> **注意**: 自動ONにすると既存の入力が上書きされることがあります。

---

## ヘルパーが対応している画面

| 画面 | ヘルパーの得意なこと |
|------|---------------------|
| **New Session** | テーマ・事前情報・ストラテジーの選び方 |
| **Personas** | ペルソナ名・ロール・事前情報の書き方 |
| **Tasks** | タスクの説明文の書き方 |

---

## ヘルパーの会話履歴

ヘルパーを開いている間は会話履歴が保持されます。
文脈を踏まえた深掘りや追加質問ができます。

ヘルパーを閉じて再度開くと、会話履歴はリセットされます。

---

## ヒント

- 具体的なテーマや目的を伝えると、より的確な提案が得られます
- 「もっと具体的に」「別のパターンで」と追加指示できます
- 提案を一度反映してから「さらに改善して」と依頼することもできます
