# New Session（セッション設定）

セッションを設定・開始するメインページです。「何について」「誰が」「どのように」議論するかをここで決めます。

## 画面の構成

### 上部タブ
- **Session Config** : テーマや事前情報を設定する
- **Participants** : 参加ペルソナとタスク割り当てを設定する

### プリセット機能
設定を名前付きで保存・読み込みできます。よく使う設定はプリセットとして保存しておくと便利です。

---

## Session Config タブ

### 共通テーマ（Common Theme）
全テーマに横断して共通する大きな方向性を記述します。

- 例：「2030年の社会課題」「新製品のマーケティング戦略」
- 個別テーマが複数ある場合に、議論の軸がぶれないよう機能します
- 省略しても構いません

### 事前情報（Pre-info）
全エージェントに共有する背景知識・前提条件を入力します。

- ドキュメントの内容・要約を貼り付けると、全エージェントがその内容を踏まえて議論します
- 制約条件・前提（「予算は100万円以内」「ターゲットは30代女性」など）も記述できます
- RAG（Data Base）を使えば、ドキュメントを自動的に参照させることもできます

### テーマ（Themes）
議論の個別トピックです。1セッションに複数設定でき、順番に議論されます。

各テーマには以下を設定できます：

| 設定項目 | 説明 |
|----------|------|
| **テーマ文** | 議論させたい問いや課題を具体的に書く |
| **参加ペルソナ** | このテーマに参加するペルソナを絞り込む（空=全員参加） |
| **出力フォーマット** | 出力の形式を指定（例：「箇条書きで3点」「Markdownで表にまとめる」） |
| **ターン数** | このテーマの最大発言ターン数（空=セッション全体のデフォルト） |
| **テーマ固有の事前情報** | このテーマだけに与える追加情報 |
| **テーマ内ストラテジー** | 議論の進め方のパターン（後述） |
| **プロジェクトフロー** | テーマ間の進行制御パターン（後述） |

---

## テーマ内ストラテジーとは

**1つのテーマ内で、エージェントがどのように発言を進めるか**を決めるパターンです。

- シーケンシャル（バトンリレー）・ブレスト・ディベートなど9種類から選択できます
- 目的に合ったストラテジーを選ぶと、議論の質が大きく変わります
- デフォルトはシーケンシャル（順番に発言）です

→ [ストラテジー一覧・選び方を見る](page:setup-strategy)

---

## プロジェクトフローとは

**複数のテーマをどの順序・条件で進行させるか**を決めるパターンです。

- ウォーターフォール・ステージゲート・アジャイルなど8種類から選択できます
- 単純に順番に実行したい場合はウォーターフォール（デフォルト）で構いません
- 品質チェックや反復改善が必要な場合は別のフローを検討してください

→ [フロー一覧・選び方を見る](page:setup-flow)

## Participants タブ

### ペルソナの選択
Personasページで作成したペルソナをセッションに追加します。

- 複数のペルソナを選択できます
- ペルソナの発言順をドラッグで並び替えられます

### タスクの割り当て

各ペルソナに「タスク」を割り当てることで、議論中の役割を与えます。

| 割り当てモード | 説明 |
|---|---|
| ランダム | セッション開始時にランダムに割り当て |
| ラウンドロビン | 順番に繰り返し割り当て |
| 固定 | ペルソナごとに個別に指定 |

---

## セッション開始

設定が完了したら **Start Session** ボタンをクリックします。
セッション開始後は Discussion 画面に自動的に移動します。

---

## ヘルパーの使い方

右下の **ヘルパー** ボタンをクリックすると、セッション設定についてAIに相談できます。
「どんなテーマを設定すればいい？」「ストラテジーの違いを教えて」など、自由に質問してください。
詳しくは [ヘルパーの使い方](page:helper) を参照してください。
