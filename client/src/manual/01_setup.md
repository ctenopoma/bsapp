# New Session（セッション設定）

セッションを設定・開始するメインページです。「何について」「誰が」「どのように」議論するかをここで決めます。

## 画面の構成

### 上部タブ
- **Session Config** : テーマや事前情報を設定する
- **Participants** : 参加ペルソナとタスク割り当てを設定する

### プリセット機能
設定を名前付きで保存・読み込みできます。よく使う設定はプリセットとして保存しておくと便利です。

---

## Session Config タブ

### 共通テーマ（Common Theme）
全テーマに横断して共通する大きな方向性を記述します。

- 例：「2030年の社会課題」「新製品のマーケティング戦略」
- 個別テーマが複数ある場合に、議論の軸がぶれないよう機能します
- 省略しても構いません

### 事前情報（Pre-info）
全エージェントに共有する背景知識・前提条件を入力します。

- ドキュメントの内容・要約を貼り付けると、全エージェントがその内容を踏まえて議論します
- 制約条件・前提（「予算は100万円以内」「ターゲットは30代女性」など）も記述できます
- RAG（Data Base）を使えば、ドキュメントを自動的に参照させることもできます

### テーマ（Themes）
議論の個別トピックです。1セッションに複数設定でき、順番に議論されます。

各テーマには以下を設定できます：

| 設定項目 | 説明 |
|----------|------|
| **テーマ文** | 議論させたい問いや課題を具体的に書く |
| **参加ペルソナ** | このテーマに参加するペルソナを絞り込む（空=全員参加） |
| **出力フォーマット** | 出力の形式を指定（例：「箇条書きで3点」「Markdownで表にまとめる」） |
| **ターン数** | このテーマの最大発言ターン数（空=セッション全体のデフォルト） |
| **テーマ固有の事前情報** | このテーマだけに与える追加情報 |
| **テーマ内ストラテジー** | 議論の進め方のパターン（下記参照） |

### テーマ内ストラテジー

| ストラテジー | 説明 | 向いている場面 |
|---|---|---|
| シーケンシャル | 順番に発言し、前の人の結果を次に渡す | 段階的に議論を積み上げたい時 |
| 並列独立（ブレスト） | 各自が独立に意見を出し、ファシリテーターが集約 | アイデア出し |
| ラウンドロビン | 全員が順番に発言するループを複数回繰り返す | 議論を深掘りしたい時 |
| 階層型 | マネージャーが計画→ワーカーが実行→評価・修正 | 品質を高めたい時 |
| 敵対的 | 生成役が提案し、批判役がダメ出しして修正を繰り返す | 提案の堅牢性を高めたい時 |
| 陪審員・裁判官 | ディベーターが議論し、裁判官が最終判定 | 複数案から最良を選ぶ時 |
| 動的ルーティング | 司会者が文脈を読んで次の発言者を動的に指名 | 柔軟な議論進行が必要な時 |
| 分割統治（Map-Reduce） | プランナーが分割→ワーカーが処理→サマライザーが統合 | 大きなタスクを分けて処理したい時 |
| 動的エージェント生成 | メタエージェントが最適なペルソナをその場で生成 | 最適な専門家が分からない時 |

### プロジェクトフロー（テーマ間の進行制御）

| フロー | 説明 | 向いている場面 |
|---|---|---|
| ウォーターフォール | テーマを順番に1つずつ実行 | 基本的な順次実行 |
| ステージゲート | 各テーマ後に品質チェック、不合格なら差し戻し | 品質基準がある時 |
| アジャイル/スプリント | 全テーマを複数回繰り返し、完成度を評価 | 反復的に改善したい時 |
| 条件分岐/ツリー | テーマの結論に応じて次のテーマを動的に選択 | 結論によって方向性が変わる時 |
| V字型 | 全テーマ実行後、逆順でレビュー | 要件と実装の整合性確認 |
| ゲーム理論/対立型 | 提案陣営vs批判陣営で議論し合意形成 | 対立する視点が必要な時 |
| ブラックボード型 | コーディネーターが共有状態を読み次の担当を指名 | 複雑な協調作業が必要な時 |
| トーナメント/進化型 | 同じテーマを並列実行し審査員が最良を選択 | 複数アプローチを比較したい時 |

## Participants タブ

### ペルソナの選択
Personasページで作成したペルソナをセッションに追加します。

- 複数のペルソナを選択できます
- ペルソナの発言順をドラッグで並び替えられます

### タスクの割り当て

各ペルソナに「タスク」を割り当てることで、議論中の役割を与えます。

| 割り当てモード | 説明 |
|---|---|
| ランダム | セッション開始時にランダムに割り当て |
| ラウンドロビン | 順番に繰り返し割り当て |
| 固定 | ペルソナごとに個別に指定 |

---

## セッション開始

設定が完了したら **Start Session** ボタンをクリックします。
セッション開始後は Discussion 画面に自動的に移動します。

---

## ヘルパーの使い方

右下の **ヘルパー** ボタンをクリックすると、セッション設定についてAIに相談できます。
「どんなテーマを設定すればいい？」「ストラテジーの違いを教えて」など、自由に質問してください。
詳しくは [ヘルパーの使い方](./08_helper.md) を参照してください。
