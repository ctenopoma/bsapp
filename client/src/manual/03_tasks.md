# Tasks（タスク管理）

議論中にエージェントに与える「役割・ミッション」を管理するページです。

## タスクとは

タスクとは、議論中にペルソナが果たす「役割・ミッション」の定義です。
ペルソナが「誰か（人物像）」を定義するのに対し、タスクは「何をするか（行動指針）」を定義します。

New Sessionのセッション設定で、ペルソナにタスクを割り当てて使用します。

**良いタスクの例：**
- 「提案の実現可能性を技術的観点から評価し、問題点を指摘してください」
- 「他の参加者の意見を要約し、共通点と相違点を整理してください」
- 「議論の結論に向けて合意形成を促し、アクションアイテムをまとめてください」

**曖昧なタスク（避けるべき）：**
- 「議論してください」
- 「意見を述べてください」

---

## 入力項目

### 説明（Description）
エージェントに何をしてほしいかを具体的に書きます。

- **具体的な行動**を指定する：「評価する」「提案する」「まとめる」など
- **視点や基準**を明示する：「技術的観点から」「コスト面で」「ユーザー視点で」
- **出力の形式**も指定できる：「3点に絞って」「箇条書きで」「賛否の理由を明示して」

**書き方のコツ：**
```
[視点・立場] から [何を] して、[どのように] 表現してください。
```

**例：**
```
財務・コスト管理の視点から各提案を評価し、
ROIと実装コストを試算した上で、優先順位をつけて推奨案を提示してください。
```

## 操作方法

| 操作 | 説明 |
|------|------|
| **Add Task** | 新しいタスクを追加 |
| **Edit（鉛筆アイコン）** | タスクを編集モードに切り替え |
| **Save（保存アイコン）** | 編集内容を保存 |
| **Cancel（Xアイコン）** | 編集をキャンセル |
| **Delete（ゴミ箱アイコン）** | タスクを削除 |

---

## ペルソナとタスクの組み合わせ方

タスクはペルソナの「ロール」と組み合わせて動作します。

| ペルソナのロール | タスク | 期待される動作 |
|---|---|---|
| 批判的なエンジニア | 技術的実現可能性の評価 | 技術リスクを厳しく指摘 |
| 楽観的なマーケター | 市場機会の提案 | 市場ポテンシャルを積極的に提案 |
| ファシリテーター | 合意形成と議論のまとめ | 意見を整理して結論を導く |

## ヘルパーの使い方

右下の **ヘルパー** ボタンをクリックすると、タスクの書き方についてAIに相談できます。
「どんなタスクを設定すればいい？」「このテーマに合うタスクを提案して」など自由に質問できます。
詳しくは [ヘルパーの使い方](./08_helper.md) を参照してください。
