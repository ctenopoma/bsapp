# Personas（ペルソナ管理）

議論に参加するAIエージェントのキャラクター（ペルソナ）を管理するページです。

## ペルソナとは

ペルソナとは、AI議論に参加するエージェントの「人物像」定義です。
名前・ロール・事前情報を設定することで、そのキャラクターとして議論に参加します。

**良いペルソナの例：**
- 批評家エンジニア：技術的な実現可能性を厳しく評価し、問題点を具体的に指摘する
- 楽観的マーケター：市場ニーズを重視し、ポジティブな可能性を積極的に提案する
- リスク管理の専門家：各提案のリスクを洗い出し、リスク低減策を検討する

---

## 入力項目

### 名前（Name）
ペルソナの名前です。議論の発言者として表示されます。

- 人物像がイメージしやすい名前をつけると議論の質が上がります
- 例：「批評家エンジニア」「楽観的マーケター」「現実主義の財務担当」
- ロールを名前に含めると分かりやすいです

### ロール（Role）
このペルソナの専門性・視点・性格を記述します。**最も重要な項目です。**

- 具体的に書くほど議論の質が上がります
- 「どの視点から」「どのように」発言するかを明確に
- 性格・口調・価値観も書けます

**良い例：**
```
あなたはソフトウェアエンジニアの視点から議論に参加します。
提案の技術的実現可能性を評価し、実装上の課題や代替手段を具体的に示してください。
楽観的な意見には冷静にリスクを指摘し、根拠のない主張には証拠を求めてください。
```

**曖昧な例（避けるべき）：**
```
エンジニアとして議論に参加してください。
```

### 事前情報（Pre-info）
このペルソナだけに与える追加知識・背景情報です。

- 特定のドキュメント内容を読み込ませたい場合に使用します
- 「このペルソナだけが知っている情報」として機能します
- 例：財務担当ペルソナには財務データを、マーケターには市場調査結果を与える

### RAG設定
このペルソナに紐づけるRAGコレクションを選択できます。
Data Baseページで登録したドキュメントを参照させることができます。

---

## プリセット機能

よく使うペルソナ設定を「プリセット」として保存できます。

- **Save as Preset** : 現在の設定を名前付きで保存
- **Load Preset** : 保存済みのプリセットを読み込む
- プリセットはNew Sessionページからも参照できます

---

## 操作方法

| 操作 | 説明 |
|------|------|
| **Add Persona** | 新しいペルソナを追加 |
| **Edit（鉛筆アイコン）** | ペルソナを編集モードに切り替え |
| **Save（保存アイコン）** | 編集内容を保存 |
| **Cancel（Xアイコン）** | 編集をキャンセル |
| **Delete（ゴミ箱アイコン）** | ペルソナを削除（確認ダイアログあり） |

---

## ヘルパーの使い方

右下の **ヘルパー** ボタンをクリックすると、ペルソナの作り方についてAIに相談できます。
「どんなロールを書けばいい？」「このテーマに合うペルソナを提案して」など自由に質問できます。
詳しくは [ヘルパーの使い方](./08_helper.md) を参照してください。
