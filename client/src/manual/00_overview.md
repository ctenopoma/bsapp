# AI Discuss マニュアル

AI Discuss は、複数のAIエージェントが議論を行うマルチエージェントディスカッションプラットフォームです。

## アプリの全体像

```
Personas（ペルソナ）  ──┐
Tasks（タスク）        ──┼──► New Session（セッション設定）──► Discussion（議論実行）
Data Base（RAG）       ──┘
```

各エージェントに「ペルソナ」と「タスク」を割り当て、テーマに沿ってAI同士が議論します。RAGを使えばドキュメントの内容を議論に反映させることもできます。

## 主なページ

| ページ | 説明 |
|--------|------|
| **New Session** | 議論セッションを作成・開始するメインページ |
| **Personas** | 議論に参加するAIエージェントのキャラクター管理 |
| **Tasks** | エージェントに与えるミッション・役割の管理 |
| **Data Base** | RAG（検索拡張生成）用のドキュメントDB管理 |
| **Patent Research** | 特許調査・分析ワークフロー |
| **Settings** | アプリ全体の設定 |

## ヘルパー機能

**New Session・Personas・Tasks** の各ページには、右下に「ヘルパー」ボタンが表示されます。
ヘルパーはAIアシスタントで、入力項目の書き方についてチャットで相談できます。
詳しくは [ヘルパーの使い方](./08_helper.md) を参照してください。

## 典型的な使い方の流れ

1. **Personas** でAIエージェントのキャラクターを作成する
2. **Tasks** で議論時のミッションを作成する
3. **New Session** でテーマ・ペルソナ・タスクを組み合わせてセッションを設定する
4. **Start Session** ボタンで議論を開始する
5. **Discussion** 画面で議論の進行をリアルタイムで確認する
